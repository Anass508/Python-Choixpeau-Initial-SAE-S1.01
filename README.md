# 🧙‍♂️ SAÉ S1.01 : Le nouveau choixpeau (pas) magique de Poudlard

Ce projet constitue la première partie de la SAÉ S1.01 (année 2025/2026). [cite_start]L'objectif est de recréer une méthode de répartition des élèves dans les quatre maisons de Poudlard suite à la disparition du Choixpeau original[cite: 248, 253].

## 👥 Binôme
* [cite_start]**Alis Rocher** 
* **Anass Fathi**

## 📜 Présentation du sujet
Le programme analyse les réponses des élèves à un questionnaire de 4 questions portant sur :
1. [cite_start]**Gryffondor** : Courage et bravoure[cite: 261].
2. [cite_start]**Serdaigle** : Intelligence et créativité[cite: 261].
3. [cite_start]**Poufsouffle** : Loyauté et travail acharné[cite: 261].
4. [cite_start]**Serpentard** : Ambition et ruse[cite: 261].

[cite_start]L'élève est affecté à la maison où il a obtenu son meilleur score[cite: 270].

## 💻 Contenu technique
* [cite_start]**`SAE_Anass Fathi_Anis Rocher_v2.ipynb`** : Notebook contenant le code principal, les calculs de statistiques et la comparaison avec une méthode de répartition aléatoire[cite: 345].
* **`questionnaire_premiere_annee.txt`** : Données d'entrée des élèves[cite: 330].
* [cite_start]**`affectation_premiere_annee.json`** : Données de référence du Choixpeau pour calculer le taux d'erreur[cite: 332].
