# 🧙‍♂️ SAÉ S1.01 : Le nouveau choixpeau magique de Poudlard

Ce projet constitue la première partie de la SAÉ S1.01 (année 2025/2026). L'objectif est de recréer une méthode de répartition des élèves dans les quatre maisons de Poudlard suite à la disparition du Choixpeau original.

##  Binôme
* **Anass Fathi**
* **Anis Rocher**

##  Présentation du sujet
Le programme analyse les réponses des élèves à un questionnaire de 4 questions portant sur :
  1. **Gryffondor** : Courage et bravoure.
  2. **Serdaigle** : Intelligence et créativité.
  3. **Poufsouffle** : Loyauté et travail acharné.
  4. **Serpentard** : Ambition et ruse.

L'élève est affecté à la maison où il a obtenu son meilleur score.

##  Contenu technique
  * **`SAE_Anass Fathi_Anis Rocher_v2.ipynb`** : Notebook contenant le code principal, les calculs de statistiques et la comparaison avec une méthode de répartition aléatoire[cite: 345].
  * **`questionnaire_premiere_annee.txt`** : Données d'entrée des élèves[cite: 330].
  * **`affectation_premiere_annee.json`** : Données de référence du Choixpeau pour calculer le taux d'erreur[cite: 332].

##  Évaluation et Qualité
Conformément aux attendus:
- Code documenté avec des **docstrings**.
- Présence de **tests unitaires** pour chaque fonction logique.
- Analyse du pourcentage d'erreur par rapport à la répartition officielle.
